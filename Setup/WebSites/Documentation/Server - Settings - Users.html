Server - Settings - Users

Server users allow you to configure who is allowed to connect and administer the server and all of it's features. From here you can also enable and disable remote administration.

Users

Allows the server administrator to add, edit, delete and reorder server users.


Setting Description
Allow remote management Enables or disables remote administration. If you eanble this option, you should ensure that all of your users have secure passwords.

Users - Item




Setting Description
Username The name of the user - this is what the user will need to nenter when logging in to administer this server.
Password The password of the user. This password is stored encrypted an Base64 encoded for security and portability within XML.
