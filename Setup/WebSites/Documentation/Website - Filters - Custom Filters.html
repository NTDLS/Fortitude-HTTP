Website - Filters - Custom Filters

Custom filters are an advanced part of server functionality which allow a developer / administrator to perform low-level filtering, auditing, logging and even dynamically alter request specifics. As of Fortitude HTTP version 1.0.3.8, custom filters have not been fully implemented and are not yet ready for mainstream use.

Custom Filters

Allows the server administrator to add, edit, delete and reorder website custom filters.
