Global vs Local

All settings on each website can be set as a per-website level or they can be set globally.
Global settings are not necessarily inherited by all websites as each website can be set to use local or to use global settings. When a site is set to use global settings and is then set to use local settings, the global setting act as the default settings for the website and the global configuration is copied to the website. When a website is using local settings and is then set to use global setting, the local settings are discarded in favor of the global settings.

Local settings branch out under each website, while global settings branch out from unser the Global Settings tree item.


The check-box "Use global values" is used on each of the configuration dialogs to toggle between local and global settings.
