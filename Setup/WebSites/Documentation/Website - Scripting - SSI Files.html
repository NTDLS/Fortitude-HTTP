Website - Scripting - SSI Files

Server Side Includes (SSI) is a simple interpreted server-side scripting language used by web servers The most frequent use of SSI is to include the contents of one or more files into a web page on a web server. For example, a web page containing a daily quotation could include the quotation by placing the following code into the file of the web page: <!--#include virtual="../quote.txt"-->

SSI Files

Allows the server administrator to add, edit, delete and reorder website SSI files/extensions.
