Website - Filters - Extension Filters

Extension filters are used to reject request for specified file extensions / types. These filters are an important part of the server's security as they (by default) disallow access to a few types which could lead to unnecesary discovery of server configuration information (such as types: XML, Log, ini). It is not advisable to disable this security feature. Any file extension/type matches will cause the server to reject the request by returning HTTP error 401.

Extension Filters

Allows the server administrator to add, edit, delete and reorder website extension/file-type filters.
