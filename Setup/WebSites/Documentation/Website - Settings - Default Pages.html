Website - Settings - Default Pages

Default pages are used to define a list of file names to search requested directories. For instance, if the connected peer requested "/images/", instead of returning an error to the peer the server will search this directory (for example) for a file named "default.html" and deliver it instead of an error.

Default Pages

Allows the server administrator to add, edit, delete and reorder website default pages. Default pages are searched for in the order specified when no file name is otherwse specified by the peer.


Default Pages - Item




Setting Description
Page name Defined the name of the file to search for in directories when no file name is otherwise specified.
