Website - Filters - IP Filters

IP filters allow peers to be blocked at a relatively low-level thereby allowing the server administrator to block peers without adding much additional overhead. When a remote peer is disconnected, they are sent an HTTP error 403.6. Similar functionality can be accomplished through the use of firewall rules.

IP Filters

Allows the server administrator to add, edit, delete and reorder website IP filters.


IP Filters - Item




Setting Description
IP Address Specifies the IP address which will be disconnected and sent an HTTP 403.6 error. Note: The IP address must be an exact match as the server does not curntly support masking.
