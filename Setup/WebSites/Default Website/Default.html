Welcome to the home page for v
Please note that this site is locked down by default and will not render much more than static HTML & images until you allow a few extensions or install at least one scripting engine such as Python, PERL or PHP, ect...


If you would like to view more of the built-in content, you will need to view the local "Examples and Tests" website.

Important: The "Examples and Tests" website mentioned above is not started by default. If you would like to view it, you will need to open the Management Console and start it.
    Accessing the Management Console:
  1. Click Start -> NetworkDLS -> Fortitude HTTP -> Management Console
  2. The Fortitude HTTP Management Console will appear.
